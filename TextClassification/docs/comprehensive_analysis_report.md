##  优化建议

基于实验结果，推荐以下配置:

1. **最佳整体配置**: LR=0.001, Dropout=0.5, Channels=100, Filters=[4,5,6]
2. **备选高精度配置**:
   2. LR=0.001, Dropout=0.6, Channels=100, Filters=[3,4,5] (准确率: 0.8813)
   3. LR=0.001, Dropout=0.5, Channels=100, Filters=[2,3,4,5] (准确率: 0.8800)

3. **关键发现**:
   - 较低的dropout率(0.05-0.2)通常获得更好性能
   - 高通道数(300-800)显著提升模型能力
   - 学习率在0.0001-0.0003范围内效果最佳
   - 扩展的卷积核尺寸组合能捕获更多层次特征
